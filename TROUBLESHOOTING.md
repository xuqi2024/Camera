# 故障排除指南

## 常见问题

### 1. 摄像头访问错误

症状：
- 无法打开摄像头
- 权限错误

解决方法：
```bash
# 检查设备
ls -l /dev/video*

# 添加权限
sudo usermod -a -G video $USER
```

### 2. 模型加载失败

症状：
- ONNX Runtime错误
- 模型文件不存在

解决方法：
- 确认模型文件路径
- 检查模型版本兼容性
- 验证预编译库安装

### 3. GUI显示问题

症状：
- Qt错误
- 显示黑屏

解决方法：
- 检查Qt安装
- 验证OpenCV编译选项
- 确认显示驱动
